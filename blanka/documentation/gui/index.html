Run BLANKA GUI

  1. Run run_blanka_gui.bat file to start BLANKA GUI.
  2. Click on the ... box next to the Sample or Control lines to select files/directories to be processed. If Multiple Sample Files/Multiple Control Files is unchecked, you will be prompted to select a single file. If Multiple Sample Files/Multiple Control Files is checked, you will be prompted to select a directory containing one or more files. Files that are not in .mzXML format will be converted to .mzXML format using MSConvert.

  3. Choose the type of data (LCQ, QTOF, or MALDI-TOF dried droplet) that will be procesed under the Experiment dropdown menu
  4. Change any other parameters relating to thresholds, tolerances, and any other parameters as desired. For a full description of all parameters, refer to the arguments table.
  5. Click Run to start processing your data.
  6. If any user specified parameters are invalid, a pop-up window will tell users which parameters have been entered incorrectly. Re-enter correct parameters and click Run again to start processing your data.
  7. If the Verbose option was selected, a new window will appear with progress on data processing.
  8. Once complete, a new window will pop up telling the user that BLANKA has completed processing.
  9. Data will be found in the directory containing the original sample files or the user specified output directory.
