Download BLANKA

  1. Download BLANKA here.
  2. Unzip the files to the desired directory.
  3. BLANKA has no installation files and is ready to go after unzipping. Users only need to configure the path to Proteowizard's MSConvert if file conversion is required. To use BLANKA GUI, Python 2 must be added to PATH. Instructions for both can be found below.

Check if Python 2 is Set to PATH

Windows 10

  1. Open Command Prompt.
  2. Enter python into the terminal.
  3. If Python 2 is already added to PATH, it should now start and you should see the following text below:
    4. Python 2.7.17 |Anaconda, Inc.| (default, May 1 2018, 18:37:09) {MSC v.1500 64 bit (AMD64)] on win32
    Type "help", "copyright", "credits" or "license" for more information.
    >>>

    Note: Information after Python version may vary based on PC and version of Python installed.
  4. If Python 2 is not added to PATH already, you will see an error message:
    5. 'python' is not recognized as an internal or external commmand, operable program or batch file.
  5. Continue to the next section if you see an error message to add Python 2 to PATH.

Add Python 2 to PATH

Windows 10

  1. Open the System Information from the Control Panel.
    2. Start > Control Panel > System and Security > System
  2. On the left side, click on Advanced system settings.
  3. Go to the Advanced tab.
  4. Click on Environment Variables... to open the Environmental Variables settins window.
  5. Under User variables for [username] highlight Path and click Edit... to open the Edit environment variable window.
  6. Click New and add the file path to python.exe. For example, the default installation of Python 2 is shown below:
    7. C:\Python27\python.exe
  7. Click OK here and in all previous settings windows to save your changes.
  8. Follow the instructions in the above section to ensure Python 2 has been correctly added to PATH.

Configure MSConvert Path

  1. By default, BLANKA will look for msconvert.exe in the default installation path:
    2. C:\Program Files\Proteowizard\Proteowizard [version number]\msconvert.exe
  2. Alternatively, a custom path to msconvert.exe can be used by adding to the path to the config.ini file if Proteowizard is not installed to the default directory:
    3. [msconvert]
    path=C:\Users\[username]\Desktop\pwiz\msconvert.exe
