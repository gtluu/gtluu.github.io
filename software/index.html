Software

TIMSCONVERT

TIMSCONVERT is a workflow that allows for the conversion of raw data from Bruker timsTOF Pro, fleX, and SCP mass spectrometers (i.e. .d directory containing BAF, TDF, and TSF files) to open source data formats (i.e. mzML and imzML) that (1) are compatiblee with downstream open-source data analysis platforms, (2) incorporate trapped ion mobility spectrometry (TIMS) data into these open-source formats, and (3) possess manageable file sizes.

  • Use TIMSCONVERT

BLANKA2

Oftentimes, media components are found in crude extracts generated for metabolomics experiments. For example, cheese curd agar is a very nutrient rich medium containing high amounts of salts, lipids, and casein breakdown products that can convolute statistical analyses. BLANKA is a tool developed in Python 2 to filter out high background signals from mass spectrometry data. By removing these background signals, we can better prioritize metabolites of interest.

  • Use BLANKA2

pyMALDIproc

pyMALDIproc is a collection of preprocessing functions for MALDI-TOF MS data seeking to bring similar functionality found in the R package MALDIquant to the Python programming environment. Includes classes for data import from mzML formats, wrapper functions to perform preprocessing, and an example dashboard for basic preprocessing and data visualization.

  • Use pyMALDIproc

cardinalscripts

cardinalscripts is a small collection of scripts and functions written to streamline data processing/analysis using the Cardinal R package from Bioconductor.

  • Use cardinalscripts

rspeclust

rspeclust is an R adaption of the SPECLUST developed by Alm et al. at Lund University, which was developed to cluster peak lists. Currently, only peak list combining and creation of consensus peak lists has been adapted.

  • Use rspeclust
